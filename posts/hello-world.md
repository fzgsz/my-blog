## 为什么写博客？

一直想有一个属于自己的角落，记录学习过程中的思考、踩过的坑、以及偶尔的灵光一闪。

之前总是想着"等准备好了再开始"，后来发现 **开始本身就是最好的准备**。

## 为什么是这个风格？

作为一个程序员，终端是每天待得最久的地方。黑色的背景、绿色的光标、等宽字体——这些元素让人感到安心和专注。

所以博客也做成了这样：

- 深黑背景 + 霓虹绿强调色
- 等宽字体标题
- 终端风格的交互细节
- 打字机动画

> 代码是诗，终端是画布。

## 技术选型

没有用任何框架，纯 HTML + CSS + JS：

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My Blog</title>
  </head>
  <body>
    <p>Hello, World!</p>
  </body>
</html>
```

简单、纯粹、没有依赖。
